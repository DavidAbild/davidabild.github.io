CSS References

Property Description Example/values
color Foreground color
background-color Background color
background-image Sets an image as the background background-image: url("paper.gif");
display Determines the display behavior none, inline, block, inline-block, flex (and more)
width Width
height Height
min-width Minimum width
min-height Minimum height
max-width Maximum width
max-height Maximum height
margin Shorthand to set margin around element margin: 10px 5px 15px 20px;
padding Shorthand to set padding inside of the element padding: 10px 5px 15px 20px;
border Shorthand to set border around element border: 5px solid red;
border-color Color of the border
border-width Width of the border
border-style Sets the style of the border dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden
border-radius Radius of the border, used to round the edges border-radius: 50% (this example will make a square element become a round circle)
font-family Sets the font family
font-style Style of the font normal, italic, oblique, initial, inherit
font-weight Determines how thick/thin characters in text should be displayed
position Positioning method used static, absolute, fixed, relative, sticky, initial, inherit
z-index Specifies the stack order of an element z-index: 10; (greater stack order is always in front of an element with a lower stack order, negatives allowed)
